# Website1

This is a basic HTML website for learning and practice.

HTML Document Structure

Understanding <!DOCTYPE html>, <html>, <head>, and <body> tags.

Text Formatting
Using headings (<h1> to <h6>), paragraphs, and alignment attributes to structure content.

Links and Images
Adding hyperlinks using <a> tags and inserting images using <img> with attributes like src, alt, height, and width.

Lists
Creating both ordered (<ol>) and unordered (<ul>) lists, and using classes for styling.

Tables
Designing tables with <table>, <thead>, <tbody>, rowspan, colspan, and captions to display structured data.

HTML Comments
Using <!-- --> to include non-displaying notes in the code for clarity and future reference.

Emmet Abbreviations
Practicing Emmet-style shortcuts (like ul>li\*3, div+p) to speed up HTML coding in editors like VS Code.
